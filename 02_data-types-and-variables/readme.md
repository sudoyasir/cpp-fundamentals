# Data Types and Variables

In C++, variables are containers for storing data values.  
Each variable must be declared with a specific data type.

## Common Data Types:
- `int`: Stores integers (whole numbers), e.g., 25, -10.
- `float`: Stores decimal numbers with single precision, e.g., 3.14.
- `double`: Stores decimal numbers with double precision, e.g., 9.81.
- `char`: Stores a single character, e.g., 'A', 'z'.
- `bool`: Stores a boolean value (`true` or `false`).
- `string`: Stores a sequence of characters (text). *(`string` requires `#include <string>`)*

## Code Explanation:
- We declare different types of variables.
- We use `cout` to display the value of each variable.

## Output:
Integer: 25 Float: 3.14 Double: 9.81 Character: A Boolean: 1 String: Hello
